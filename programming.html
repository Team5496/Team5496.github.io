About Programming

Members of the programming department are responsible for writing the code to control the robot as well as setting up the driver's station for the drivers before each match. The programming department doesn’t require any previous coding knowledge but if you're a beginner and would like to get a head start you can start by researching the Java programming language (that's what our team uses).

Getting started

In order to get started you will need to install a few things, luckily the WPILib installer makes this relatively easy by setting up all the basics for you. To download the WPILib installer, click here and scroll until you see a link starting with “WPILibInstaller_Windows64” and ending with “.zip” then click the link. Once the WPILib installer has downloaded, extract the zip file and run the installer. Select either “All Users” or “Current User” (“All Users” requires administrator access). Next click “Select/Download VS Code” then “Download” to install VS Code. After installing VS Code, click “Execute Install”. This will install WPILib, Java, and a few other necessary tools. Once the installer has finished, there should be an FRC VS Code icon on the desktop. In order to create a new project, start VS Code then click on “View” -> “Command Palette...” or press Ctrl + Shift + P to open the command palette. Begin typing “Create a new project” and click on the option that says “WPILib: Create a new project”. This will open the project creator where the details of the new project can be entered. For our team's robot projects, the first three options should be “template”, “java”, and “Timed Robot”. Click on each option button to change it.
